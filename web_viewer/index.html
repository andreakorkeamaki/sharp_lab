3DGS Viewer

Load a local SHARP `.ply` or a hosted splat URL. This viewer is static and can be run locally or deployed to Vercel later.

Best for local tests. Uses the browser file API, so the splat never has to leave your computer.

The remote file must allow browser access with CORS headers.

Viewer

  • Left drag: orbit
  • Right drag: pan
  • Wheel: zoom

Orientation

SHARP exports in a camera-centric OpenCV convention. Some viewers need a manual flip or turn to line the splat up with web-style coordinates.
